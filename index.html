VPSDeploy: 一键部署企业级安全网络节点

License: MIT version 1.0 OS Debian | Ubuntu

在当今复杂的网络环境中，拥有一个私密、稳定且高效的个人网络节点至关重要。VPSDeploy 是一个为开发者、跨境工作者和DIY爱好者设计的开源工具，旨在解决部署难题，让您只需一行命令，即可在全新的服务器上部署当前最先进的 Xray (VLESS + TCP + XTLS-Reality) 安全节点。

核心优势

  • 一键化脚本: 告别繁琐的手动配置。整个过程全自动，无需任何交互，脚本执行完毕即可使用。
  • 最新技术方案: 采用最新的 VLESS + XTLS-Reality 协议。这套组合在保证强大性能的同时，拥有顶级的伪装能力，能有效应对复杂的网络环境。
  • 智能环境检测: 脚本在执行前会自动检测服务器的操作系统和运行权限，提前规避常见的部署错误，大大提高成功率。
  • 开源透明: 所有代码均在 GitHub 公开，确放心使用，数据完全由您自己掌控。

🛠️ 准备工作

在开始之前，您只需要准备一样东西：

  • 一台位于海外的、全新的、纯净的 Debian (11+) 或 Ubuntu (20.04+) 操作系统的服务器 (VPS)。

重要提示: 请勿在已经安装了其他网站或服务的服务器上运行此脚本，以避免端口冲突或环境污染。

一行命令，即刻部署

通过 SSH 登录到您的服务器，然后复制并执行以下命令：

wget -P /root https://raw.githubusercontent.com/SongZstrong/VPSDeploy/main/VPSDeploy.sh && sudo bash /root/VPSDeploy.sh
